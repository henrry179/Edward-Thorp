# 输出文件目录

本目录用于存放脚本和 Notebook 运行后生成的所有输出文件。

## 目录结构

- `figures/`：图片文件（PNG 格式，300 DPI）
  - 所有脚本和 Notebook 生成的图表都会自动保存到此目录
  - 文件名格式：`{脚本名}_{描述}_{时间戳}.png`

- `data/`：数据文件（CSV 格式，UTF-8-BOM 编码）
  - 回测结果、敏感性分析数据、信号数据等
  - 文件名格式：`{数据类型}_{时间戳}.csv`

- `reports/`：报告文件（预留）
  - 用于存放生成的报告文档

## 文件命名规则

所有输出文件都包含时间戳（格式：`YYYYMMDD_HHMMSS`），确保：
- 多次运行不会覆盖之前的文件
- 可以追踪文件生成时间
- 便于版本管理和对比分析

## Git 跟踪

本目录中的文件默认会被 Git 跟踪（除非在 `.gitignore` 中排除），这样可以：
- 保留历史运行结果
- 便于在 GitHub 上展示项目输出
- 方便复现和对比分析

如需排除某些文件类型，可在 `.gitignore` 中添加相应规则。
